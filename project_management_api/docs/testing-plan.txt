AUTH
Endpoint: POST /api/auth/signup
Access Control: Public

Success Case:
Enter body:
  	{
    "email": "user5@test.com",
    "password": "password5",
    "role": "USER"
  }
Click Try it out.
Click Execute.
Expect 201 Created.

400 Bad Request:
Use invalid email or short password.
Click Try it out.
Click Execute.
Expect 400 Bad Request.

Endpoint: POST /api/auth/login
Access Control: Public (rate-limited)

Success Case:
Enter body:
  {
    "email": "user5@test.com",
    "password": "password5"
  }
Click Try it out.
Click Execute.
Expect 200 OK with accessToken.

401 Unauthorized:
Use wrong password.
Click Try it out.
Click Execute.
Expect 401 Unauthorized.

429 Too Many Requests:
Submit invalid login 4+ times in one minute.
Click Try it out.
Click Execute.
Expect 429 Too Many Requests.
